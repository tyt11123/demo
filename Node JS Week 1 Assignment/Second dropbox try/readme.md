# Project Title

Weekly Assignment - Dropbox with Cache

## Getting Started

This is a Dropbox with cache. An entry page is provided for uploading, and an index page is provided to show all the stored files.

### Prerequisites

Node.js
express
fs
path
express-fileupload

## Deployment

Run node app.js or nodemon app.js in Terminal and access the entry page
